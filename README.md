# OnlyOneOnlyBird
* Kotlin実装のルーレットアプリ
* 設定された名称のリストから１名を抽選できます
![画面](./view.png)

## Hwo to use
* gradleをインストールしている場合

```bash
clone 
cd OnlyOneOnlyBird
gradle build
gradle run
```
* gradleをインストールしていない場合は、gradleをインストールするか、プロジェクト内のgradlewを使ってビルド実行可能です。

```bash
clone 
cd OnlyOneOnlyBird
./gradlew build
./gradlew run
```

## Setting
* resouces/setting/name-list.txtファイルにある名簿が抽選対象者
